# How to Download & Run VoxStat Overlay

### Download
> **1.** Download the `VoxStatsOverlay.zip`.
> **2.** Extract the zip file.
> **3.** Open the extracted VoxStatOverlay folder.
> **4.** Double-click `Overlay.exe` to start.

*Do not move the `.exe` out of the folder. The application needs the other files in the folder to run correctly.*

### First time setup
> **1.** Click the ⚙️ Settings button.
> **2.** Select your Minecraft `latest.log` file.
> **3.** Enter your Voxyl API key.
> **4.** Click Save.

*You can get your Voxyl API key by running `/api new` or `/api get` in game.*
